  • setParticipantHidden(participantId: string, isHidden: boolean): void
    • Defined in src/helpers/sceneless-project.ts:382

    Hide a participant video feed from the stream. This does not affect the underlying MediaStreamTrack.

    Participants hidden in this way will not stop sending video data, but it will not play on the receiving end.

    A host may use this to override a guest's settings for the stream output.

    Parameters

    • participantId: string
    • isHidden: boolean

    Returns void
